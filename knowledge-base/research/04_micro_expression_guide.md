# 微表情（Micro-expressions）検出ガイド
> 出典：Micro-expressions: A Study of Basic Reading and The...（Yang論文）
> 動画フレーム解析AIへの指示書として使用する。

---

## 1. 微表情が発生するメカニズム

人が自身の真の感情を**隠そうとしたり抑圧しようとした際に、意図せず無意識に漏れ出る瞬間的な顔の表情**。意識的に感情をコントロールしようとする行動が、結果として顔面筋肉の不随意な動きを引き起こすことで発生する。

- 持続時間：0.5秒〜最長4秒（一部は数分の1秒で消え去る）
- 人間の目では見逃されがちなため、フレーム単位での精緻な解析が必須

---

## 2. 7つの基本感情の観察ガイド

> ※ 解剖学的な筋肉名が明示されているのは「喜び」のみ。他は観察可能な顔の部位の動きとして定義。

### ① 喜び（Happiness / Joy）
| 項目 | 内容 |
|------|------|
| 出現部位・筋肉 | 大頬骨筋（口角を頬骨に向かって引き上げる）＋眼輪筋（頬を持ち上げ、目の周りに「カラスの足跡」のシワを作る） |
| 典型的な文脈 | 好印象を与えようとする際の自己提示、または嘘をつくことに成功しそうな際の「だます喜び（Duping Delight）」 |
| 読む際の注意点 | **作り笑い（False smiles）との識別が重要。** 作り笑いは眼輪筋の動きが欠如し、顔の動きが非対称で、出現タイミングが早すぎ/遅すぎ、不自然に長く持続する |

### ② 悲しみ（Sadness）
| 項目 | 内容 |
|------|------|
| 出現部位 | 口元：口角が下がる（Lips: Corners down） |
| 典型的な文脈 | 過去のネガティブな経験や失敗について語る際、または同情を引こうとする際（Supplication） |
| 読む際の注意点 | 真実を語る者が無実の罪を疑われて悲しむケースとの混同に注意 |

### ③ 怒り（Anger）
| 項目 | 内容 |
|------|------|
| 出現部位 | 眉間：シワを寄せる・眉をひそめる（Eyebrows: Frown）、顔をしかめる（Scowl） |
| 典型的な文脈 | 面接官からの厳しい質問や、自身の矛盾を指摘されて防衛的・攻撃的になった際に瞬間的に漏れ出る |
| 読む際の注意点 | 感情の文脈（前後関係）に強く影響される。ポジティブな気分の状態にある時に怒りが引き起こされると、より強く現れる傾向がある |

### ④ 恐怖（Fear）
| 項目 | 内容 |
|------|------|
| 出現部位 | 目：大きく見開く（Eyes: Exaggerated opening）、または両目を閉じる（Close both eyes） |
| 典型的な文脈 | 嘘が露見するリスクを感じた際や、高リスクな面接状況下で強い不安を感じた際 |
| 読む際の注意点 | **恐怖＝嘘ではない。** 真実を語る者も高いプレッシャーや疑われる恐怖により同様の反応を示す。ノイズになりやすいため単独での判定は禁止 |

### ⑤ 驚き（Surprise）
| 項目 | 内容 |
|------|------|
| 出現部位 | 眉：上げる（Eyebrows: Raise）、口：開ける（Mouth: Open） |
| 典型的な文脈 | 予期せぬ質問をされた際や、面接官が検証可能な証拠を不意に提示した際 |
| 読む際の注意点 | 驚きは単独よりも「恐怖」や「怒り」などと融合した複合的な表情（Amalgamations）を形成することが多い |

### ⑥ 嫌悪（Disgust）／ ⑦ 軽蔑（Contempt）
| 項目 | 内容 |
|------|------|
| 出現部位 | 顔をしかめる（Scowl）、非対称な口元の動き（滑らかさの欠如） |
| 典型的な文脈 | ダークトライアド（特にマキャベリアニズム）特性を持つ人物が、他者の動機を冷笑的に語る際や他者を見下す際（Misanthropic identity） |
| 読む際の注意点 | 「嘘をついている」というサインではなく、対象者のベースにある「他者への不信感」や「操作的な性格特性」の漏出である可能性が高い |

## 3. 面接文脈で特に注意すべき組み合わせパターン

### 微表情の希少性に注意
Porterらの研究では、微表情は動画全体のわずか**2%**でしか確認されず、**嘘つきよりも「真実を語る者」の方が微表情を多く表示した**という結果もある。微表情の存在自体を即座に「嘘の証拠」とスコアリングしてはいけない。

### 滑らかさと非対称性（Smoothness and asymmetry）
表情を偽装しようとする場合、顔の動きの「滑らかさの欠如」や「左右非対称性」が生じる。AIの検知アルゴリズム（幾何学的特徴の抽出）に組み込むことを推奨。

### マルチモーダルでの矛盾検知（最重要・d=0.34）
微表情単体よりも、**発言内容と顔・身体の動きが矛盾している状態**をマルチモーダルで検知することが精度の高い指標となる。
- 例：「いいえ」と否定する発言 ＋ 首を縦に振る
- 例：深刻な話をしている最中の瞬間的な薄笑い（Duping Delight）

## 4. 微表情を見逃しやすい条件と対策

| 条件 | 詳細 | 対策 |
|------|------|------|
| 持続速度 | 0.5秒未満〜数分の1秒で発生・消失 | フレーム単位での精緻な解析（ハイスピードカメラ・コンピュータービジョン）が必須 |
| 撮影環境 | 照明・角度・画質が不十分だと検出困難 | 対象者がカメラの真正面、顔全体が明確に見える高画質映像が必須条件 |
| 文化的差異 | 欧米は顔全体で感情表現（可動域が広い）。東洋の文化圏（集団主義的・内向的）では目の周りに偏り、全体的な表情の動きが小さい | 対象者の文化圏に応じて「目の周辺」への重み付けを変える |
| 身体状態 | 人混み・緊張状態・身体が動いている状態では発生頻度や見え方が変化する | 対象者固有のベースライン（普段の表情の動き）を最初に把握してから比較判定する |
