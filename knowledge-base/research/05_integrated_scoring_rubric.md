# 統合評価スコアリングルーブリック
> 出典：4論文横断（Vrij論文・Pérez-Rosas論文・Ruben David論文・Yang論文）
> 面接動画を6軸・0〜100点でスコアリングするための基準。

---

## Axis 1：言語的誠実性（回答の具体性・一貫性・詳細度）

**根拠論文：** Vrij論文・Pérez-Rosas論文・Ruben David論文
**エビデンス強度：強**（言語的詳細さの欠如 d=-0.30、直接性の低下 d=-0.55 は最大の識別指標）

| スコア | 該当する観察事実 |
|--------|---------------|
| 80〜100点（低リスク） | 自身の直接的・個人的な経験として語る度合い（Immediacy）が高く、豊富で具体的な詳細（Details）が含まれる。論理的構造が一貫しており、もっともらしさ（Plausibility）がある。不必要に強い断定語を使わず自然な文脈で説明できている |
| 40〜79点（中リスク） | 多少の抽象的な表現や詳細の省略が見られるが、全体の辻褄は合っている。「えー」「あー」などの言い淀みや沈黙があるが、これらは嘘と無関係（d=0.02〜0.04）であるため減点対象としない |
| 0〜39点（高リスク） | 話の詳細さが著しく欠如し論理的構造が破綻している。または「絶対に」「完全に」等の強い確信語を多用しながら内容は抽象的で三人称（「彼ら」「それ」）を多用し心理的距離を置こうとする。他者の手柄を自分のものとして語る（Borrowing）や存在しないスキルをでっち上げる（Inventing）兆候がある |
| ⚠️ レッドフラグ | 面接官が客観的に検証可能な詳細（日時・同席者など）を求めた途端に、回答が極端に曖昧になるかはぐらかす |

## Axis 2：非言語的自然さ（言語内容と身体反応の一致度）

**根拠論文：** Vrij論文
**エビデンス強度：中**（矛盾・両価性 d=0.34、作り笑いや意図的アイコンタクトの識別有効性）

| スコア | 該当する観察事実 |
|--------|---------------|
| 80〜100点（低リスク） | 発話を補う自然な身振り手振り（Illustrators）が豊富に出現する。笑顔の際、口元だけでなく目元の筋肉（眼輪筋）も動く本物の笑顔（Felt smiles / Duchenne smiles）である。長期記憶を検索する際、自然に視線を外す（非視覚的サッカード） |
| 40〜79点（中リスク） | 面接の緊張からくるそわそわした動き（Fidgeting）や姿勢の変更（Posture shifts）が見られる。これらは真実を語る者にも等しく発生するため中程度のリスクとする |
| 0〜39点（高リスク） | 言語内容と表情・声のトーンとの間に明確な矛盾・両価性（Ambivalence）が生じている。相手が嘘を信じているか監視するための不自然に長い凝視（意図的アイコンタクト）。目元が笑っていない非対称でタイミングが不自然な「作り笑い（False smiles）」が見られる |
| ⚠️ レッドフラグ | 否定的な言葉（「やっていない」など）を発しながら無意識に頷くなど、音声と身体反応が完全に不一致を起こしている |

## Axis 3：認知的負荷サイン（嘘による頭の使いすぎを示す指標の有無）

**根拠論文：** Vrij論文
**エビデンス強度：強**（認知的負荷理論に基づく非言語の減少・声の緊張 d=0.26〜0.30）

| スコア | 該当する観察事実 |
|--------|---------------|
| 80〜100点（低リスク） | 予期せぬ質問や複雑な質問に対しても、動作が極端に止まることなく自然なペースで回答が構成される。声のトーンやピッチに極端な緊張（Vocal tension）が見られない |
| 40〜79点（中リスク） | 複雑な質問に対して考え込む様子が見られ、一時的に手の動きが止まったり話す速度が落ちたりするが、記憶を辿っている自然な反応の範囲内 |
| 0〜39点（高リスク） | もっともらしい嘘をリアルタイムで構築・監視するために脳の処理リソースが枯渇し、手の動き（Illustrators）が著しく減少する。言葉や声に強い不確実性（Uncertainty）が現れ、声のピッチが不自然に上昇する |
| ⚠️ レッドフラグ | 「出来事を逆時系列で説明して」などの認知的負荷を高める質問（Cognitive load approach）を与えられた際に、話が完全に破綻するか身体の動きが完全にフリーズする |

## Axis 4：微表情リスク（隠された感情の兆候）

**根拠論文：** Yang論文（Micro-expressions）、Vrij論文
**エビデンス強度：弱〜中**（微表情の発生頻度が全体の2%程度と稀で真実を語る者も表出するため、文脈との「矛盾」がある場合のみ中程度のエビデンス）

| スコア | 該当する観察事実 |
|--------|---------------|
| 80〜100点（低リスク） | 顔の表情変化がなめらかで対称的であり、発言内容の文脈に完全に合致している。表情の出現から消失までのタイミングが自然 |
| 40〜79点（中リスク） | 高プレッシャーな面接状況下での「恐怖（Fear）」や「不安」の微表情が漏れ出る。これは嘘つきだけでなく疑われることを恐れる真実を語る者にも発生するため、必ずしも欺瞞の証拠ではない。文化的背景（東洋文化など）により表情の可動域が狭い場合がある |
| 0〜39点（高リスク） | 感情を隠蔽・抑圧しようとする際に0.5秒〜4秒未満の瞬間的で不随意な筋肉の動き（Micro-expressions）が漏れ出る。特にポジティブな発言中の「怒り（しかめっ面）」や深刻な話の中の「だます喜び（Duping Delight / 瞬間的な薄笑い）」など、文脈と相反する感情の漏出が見られる |
| ⚠️ レッドフラグ | 自身の経歴や他者に関する質問に対し、瞬間的かつ非対称な「軽蔑（Contempt）」や「嫌悪（Disgust）」の微表情が漏れ出る |

## Axis 5：ダークトライアドリスク（マキャベリアニズム等の行動特徴）

**根拠論文：** Ruben David論文
**エビデンス強度：強**（マキャベリアニズム特性はIFBと相関 r=0.45、特に道徳的柔軟性の欠如は強いリスク指標）

| スコア | 該当する観察事実 |
|--------|---------------|
| 80〜100点（低リスク） | 他者の動機を肯定的に捉え、チームの成果に対して適切に他者にクレジットを与える。自身の弱点や過去の失敗について防衛的にならず率直に語ることができる |
| 40〜79点（中リスク） | 野心やステータスへの欲求（Desire for Status）は強いが、他者を操作しようとする意図や倫理的逸脱を正当化する発言は見られない |
| 0〜39点（高リスク） | 目的達成のためなら手段を選ばない「道徳的柔軟性・操作（Amoral Manipulation）」を合理化する発言が見られる。他者に対して支配的であり（Desire for Control）、他者を見下す・謝罪しない等のミサンスロピック・アイデンティティを投影して自身を優位に見せようとする |
| ⚠️ レッドフラグ | 「人は結局自分の利益のために動く」という極度に冷笑的な人間観（Cynicism）や「チームメンバーは出し抜くもの」といった発言が漏れ出る。面接官の質問意図を過剰に探るような警戒心を示す |

## Axis 6：CWB予測リスク（アサイン後トラブルの予兆）

**根拠論文：** Ruben David論文
**エビデンス強度：強**（マキャベリアニズムの「他者への不信（Distrust of Others）」ファセットはCWBの直接的かつ強力な予測因子 β=0.508）

| スコア | 該当する観察事実 |
|--------|---------------|
| 80〜100点（低リスク） | 過去の職場環境や同僚との関係を語る際、健全な対人関係の構築が伺える。不正の機会（Opportunity）やプレッシャー（Pressure）があっても、倫理的規範（Capability / Rationalizationの否定）を保つ発言が一貫している |
| 40〜79点（中リスク） | 過去の職務における不満やストレス（CWBの文脈的要因）を語るが、それが組織への報復やサボタージュといった破壊的行動に結びついている兆候はない |
| 0〜39点（高リスク） | 最も強力なCWB予測因子である「他者への強い不信感（Distrust of Others）」が言動に表れている。自身の失敗の責任を外部や他者に転嫁し（Image Protection / Distancing）、他者の動機を常に疑っている。他者への共感性が欠如しており、対人トラブル（I-CWB）や組織への反生産的行動（O-CWB）を引き起こす心理的閾値が低い |
| ⚠️ レッドフラグ | 「同僚に裏切られるのが当然」「組織のルールは形骸化している」といった発言（不正の合理化：Rationalization）を悪びれずに語る。これは入社後の情報漏洩・サボタージュ・ハラスメントの極めて強い予兆 |
