# 参考資料層（Reference Layer）

## 概要

このディレクトリには、面談評価の理論的背景や学術的根拠となる参考資料を格納します。

**重要**: この層のドキュメントはAIプロンプトには投入されません。人間の閲覧・学習用です。
AIの評価基準として使用されるのは `criteria/` 層（現在は `presets/ses_interview.md`）のみです。

## ディレクトリ構成

```
reference/
├── README.md                      # 本ファイル
├── behavioral-psychology.md       # 行動心理学の基礎理論
├── cbca-reality-monitoring.md     # CBCA/RM/VA理論の詳細
└── (今後追加)                      # 書籍要約、社内ノウハウ等
```

## 資料の追加方法

1. Markdownファイル（`.md`）を作成してこのディレクトリに配置
2. または、Streamlitの設定画面「参考資料」セクションからアップロード
3. ファイル名は英語のケバブケース推奨（例: `interview-techniques.md`）

## 注意事項

- この層の資料はAIプロンプトに含まれないため、ファイルサイズの制約はありません
- 書籍の要約、社内面談ノウハウ、研修資料等を自由に追加できます
- 評価基準を変更したい場合は `presets/` 内のファイルを編集してください
