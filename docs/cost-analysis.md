# コスト分析

## 前提条件

### 利用環境
- **AIモデル**: Gemini 1.5 Pro（長時間の動画・大量のテキスト解析に最も適したモデル）
- **利用方法**: Google AI Studio または Vertex AI経由のAPI呼び出し

### 処理規模
- **動画データ**: 1名あたり30分（1,800秒）× 月間50名 = 月間1,500分
- **専門書（テキスト）データ**: 約10万トークン（日本語の書籍1冊分程度）を毎回動画と一緒に読み込ませる
- **出力データ**: 1名あたり1,000トークン（約1,000〜1,500文字のリスク評価レポート）× 月間50名

### API価格設定（Gemini 1.5 Pro）
- **入力トークン**: $2.50 / 100万トークン（>128kコンテキスト時）
- **出力トークン**: $10.00 / 100万トークン（>128kコンテキスト時）
- **為替レート**: 1ドル = 150円で換算

### トークン消費量の仕様
- **動画**: 1秒あたり約263トークン

## APIランニングコスト試算

### 結論
**月額 約11,000円（約72ドル）**という非常に安価なAPIランニングコストで運用可能です。

### 詳細内訳

| 項目 | トークン数（月間合計） | API利用料（月額推計） | 備考 |
|------|----------------------|---------------------|------|
| ① 動画の入力 | 約 23,670,000 トークン | 約 $59.18（約 8,877円） | 1秒あたり263トークンで計算<br>（473,400トークン × 50名） |
| ② 専門書の入力 | 約 5,000,000 トークン | 約 $12.50（約 1,875円） | 10万トークン × 50名分 |
| ③ 解析結果の出力 | 約 50,000 トークン | 約 $0.50（約 75円） | 1,000トークン × 50名分 |
| **合計** | **約 28,720,000 トークン** | **約 $72.18（約 10,827円）** | ※インフラ費用等は除く |

## 計算式の詳細

### ① 動画入力のコスト
```
動画時間: 30分 × 60秒 = 1,800秒
トークン数: 1,800秒 × 263トークン/秒 = 473,400トークン/名
月間トークン数: 473,400トークン × 50名 = 23,670,000トークン
コスト: 23,670,000 ÷ 1,000,000 × $2.50 = $59.18
円換算: $59.18 × 150円 = 8,877円
```

### ② 専門書入力のコスト
```
ナレッジ: 100,000トークン/回
月間トークン数: 100,000トークン × 50名 = 5,000,000トークン
コスト: 5,000,000 ÷ 1,000,000 × $2.50 = $12.50
円換算: $12.50 × 150円 = 1,875円
```

### ③ 出力のコスト
```
レポート: 1,000トークン/名
月間トークン数: 1,000トークン × 50名 = 50,000トークン
コスト: 50,000 ÷ 1,000,000 × $10.00 = $0.50
円換算: $0.50 × 150円 = 75円
```

## その他のコスト要素

### インフラコスト
実装方式により異なります:

#### パターンA: Google Apps Script（簡易版）
- **追加コスト**: ほぼゼロ
- Google Workspaceの既存契約内で利用可能
- Googleドライブのストレージ容量のみ考慮

#### パターンB: Vertex AI + GCP統合（本格版）
- **Cloud Storage**: 動画保存用（約 $0.02/GB/月）
  - 30分動画を500MB/本と想定: 50本 × 0.5GB = 25GB
  - ストレージコスト: 25GB × $0.02 = $0.50/月（約75円）
- **Cloud Functions**: 実行時間に応じた従量課金
  - 月間50回実行、1回あたり数十円程度
  - 推定: 月額 1,000〜2,000円程度
- **BigQuery**: データ蓄積・分析用（オプション）
  - ストレージ: $0.02/GB/月
  - クエリ: $5/TB
  - 推定: 月額 1,000円未満

**合計推定（本格版）**: 月額 2,000〜3,000円程度の追加インフラコスト

## ビジネスとしての投資対効果（ROI）

### コスト vs リスク回避効果

#### 投資額
- **AIランニングコスト**: 約11,000円/月
- **インフラコスト**: 0〜3,000円/月
- **合計**: 約11,000〜14,000円/月

#### リスク回避効果（1件のトラブルを防いだ場合）
SES事業において、アサイン後の要員トラブルや契約解除による損失:

- **直接的コスト**:
  - 代替要員の緊急手配コスト
  - 契約解除に伴う違約金リスク
  - 火消し対応のための工数（人件費）

- **間接的コスト**:
  - お客様からの信用低下
  - 案件の赤字化
  - 将来の取引機会の損失

**推定損失額**: 1件あたり 数十万円〜数百万円

### ROI計算
```
月額投資: 14,000円
年間投資: 168,000円

重大トラブル1件を防いだ場合の損失回避: 最低 500,000円

ROI = (500,000円 - 168,000円) / 168,000円 × 100
    = 197.6%

年間2件以上のトラブルを防げば、投資対効果は極めて高い
```

## コスト最適化の可能性

### 1. モデル選択の最適化
- 初期検証: Gemini 1.5 Flash（より安価）でテスト
- 精度要件に応じてProモデルに切り替え

### 2. ナレッジの効率化
- 専門書全文ではなく、重要部分のみを抽出
- トークン数を削減しつつ精度を維持

### 3. バッチ処理の活用
- 複数の動画をまとめて処理
- API呼び出し回数の最適化

### 4. キャッシング戦略
- 同じナレッジを複数回読み込まない仕組み
- Vertex AIのキャッシング機能の活用（今後の機能拡張で可能性）
