# AI選択肢の比較

## 動画解析が可能な主要AIサービス

### 1. OpenAI GPT-4o / GPT-4o-mini

#### 機能
- **動画解析**: ✅ 対応（2024年末から正式対応）
- **マルチモーダル**: 動画、画像、音声、テキストを同時処理
- **コンテキスト長**: 128,000トークン

#### 価格（2026年3月時点）
| モデル | 入力 | 出力 |
|--------|------|------|
| GPT-4o | $2.50 / 1M tokens | $10.00 / 1M tokens |
| GPT-4o-mini | $0.15 / 1M tokens | $0.60 / 1M tokens |

#### 動画処理のトークン消費
- **GPT-4o**: 約170トークン/秒（低解像度）、約765トークン/秒（高解像度）
- **推定コスト（30分動画、低解像度）**:
  - 1,800秒 × 170トークン = 306,000トークン
  - 入力コスト: $0.77（約115円）× 50名 = **約5,750円/月**

#### メリット
- **最も安価**: 特にGPT-4o-miniは圧倒的に低コスト
- **日本語精度**: 非常に高い
- **API成熟度**: 豊富なドキュメントとコミュニティ
- **Fine-tuning可能**: カスタマイズが容易

#### デメリット
- 動画処理は比較的新しい機能
- OpenAIのデータポリシー（学習利用）に注意

---

### 2. Google Gemini 1.5 Pro / Flash

#### 機能
- **動画解析**: ✅ 対応（長時間動画に強い）
- **マルチモーダル**: 動画、画像、音声、テキストを同時処理
- **コンテキスト長**: 1,000,000トークン以上（圧倒的）

#### 価格
| モデル | 入力 | 出力 |
|--------|------|------|
| Gemini 1.5 Pro | $2.50 / 1M tokens | $10.00 / 1M tokens |
| Gemini 1.5 Flash | $0.075 / 1M tokens | $0.30 / 1M tokens |

#### 動画処理のトークン消費
- **約263トークン/秒**
- **推定コスト（30分動画）**:
  - 1,800秒 × 263トークン = 473,400トークン
  - 入力コスト: $1.18（約177円）× 50名 = **約8,850円/月**

#### メリット
- **超長文処理**: 100万トークン以上のコンテキスト
- **動画処理に特化**: 長時間動画の解析が得意
- **Google Cloud統合**: GCP環境との親和性

#### デメリット
- GPT-4oより若干高コスト
- APIの日本語ドキュメントが少ない

---

### 3. Anthropic Claude 3.5 Sonnet / Opus

#### 機能
- **動画解析**: ❌ 非対応（画像のみ）
- **マルチモーダル**: 画像とテキスト
- **コンテキスト長**: 200,000トークン

#### 価格
| モデル | 入力 | 出力 |
|--------|------|------|
| Claude 3.5 Sonnet | $3.00 / 1M tokens | $15.00 / 1M tokens |
| Claude 3 Opus | $15.00 / 1M tokens | $75.00 / 1M tokens |

#### 備考
- **動画非対応のため、今回の用途には不向き**
- ただし、動画から抽出したフレーム画像の解析は可能
- テキスト解析の精度は非常に高い

---

### 4. Azure OpenAI Service

#### 機能
- OpenAI GPT-4oと同等
- Microsoftのエンタープライズ向けサービス

#### メリット
- **企業向けコンプライアンス**: データが学習に使われない保証
- **Azure統合**: 既存Azure環境との統合が容易
- **SLA保証**: エンタープライズ向けサポート

#### デメリット
- 価格がやや高い（地域により変動）
- 審査・契約プロセスが必要

## 推奨：コスト vs 性能の比較

### 月間50名・30分動画の場合

| AI | 動画入力コスト | ナレッジ入力コスト | 出力コスト | 合計（月額） | 動画対応 |
|----|---------------|-------------------|-----------|-------------|---------|
| **GPT-4o-mini** | 約**860円** | 約190円 | 約30円 | **約1,080円** | ✅ |
| **GPT-4o** | 約5,750円 | 約1,250円 | 約500円 | **約7,500円** | ✅ |
| **Gemini Flash** | 約660円 | 約94円 | 約15円 | **約770円** | ✅ |
| **Gemini Pro** | 約8,850円 | 約1,250円 | 約500円 | **約10,600円** | ✅ |
| Claude 3.5 | - | - | - | 動画非対応 | ❌ |

### 結論：最もコスパが良いのは「Gemini 1.5 Flash」

## 推奨アプローチ

### Phase 1: プロトタイプ（PoC）
**使用AI**: **Gemini 1.5 Flash**
- 理由: 最も低コスト（月額770円）で動画対応
- 精度が不足する場合はGPT-4oやGemini Proに切り替え

### Phase 2: 精度検証
- Gemini Flash vs GPT-4o で精度比較
- コストと精度のバランスを見て最終決定

### Phase 3: 本番運用
- 最適なモデルで本番運用開始
- 必要に応じてハイブリッド構成（簡易判定はFlash、詳細分析はPro）

## 実装の方向性

### アーキテクチャ変更
従来の「動画 + PDF書籍を都度送信」から、以下に変更:

```
【旧】
動画 + PDF書籍（毎回送信）→ AI → 評価結果

【新】
1. ナレッジベースの事前構築
   インターネット情報 → ドキュメント化 → ナレッジDB

2. 解析時
   動画 → AIで文字起こし + 行動分析
   ↓
   ナレッジDBと突合
   ↓
   評価結果
```

### 技術スタック（修正版）
- **AI**: Gemini 1.5 Flash（またはGPT-4o-mini）
- **ナレッジ管理**: Markdown / JSON形式でGitHub管理
- **実装**: Python + Streamlit（簡易WebUI）
- **ストレージ**: ローカル or Google Drive
