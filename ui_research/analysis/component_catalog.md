### レーダーチャート（Pymetrics風）
**採用サイト**: [分析後に記入]

**特徴**:
- 6-8軸のレーダー
- 半透明の塗りつぶし
- グリッドライン
- カテゴリラベル外側配置

**適用場面**:
- 多次元評価（6カテゴリスコア）

**実装例（Plotly）**:
```python
import plotly.graph_objects as go

def create_radar_chart(categories, scores):
    fig = go.Figure(data=go.Scatterpolar(
        r=scores,
        theta=categories,
        fill='toself',
        fillcolor='rgba(63, 131, 248, 0.3)',
        line=dict(color='#3F83F8', width=2)
    ))

    fig.update_layout(
        polar=dict(
            radialaxis=dict(visible=True, range=[0, 100])
        ),
        showlegend=False
    )

    return fig
```
